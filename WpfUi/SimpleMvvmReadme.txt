Welcome to the NuGet package for Simple MVVM Toolkit!

The NuGet version of the Simple MVVM Toolkit contains the minimum set of assemblies
you need to get started building MVVM applications for Silverlight, WPF or Windows Phone.

The full version of the Simple MVVM Toolit includes Visual Studio item and project templates, 
code and xml snippets, as well as numerous samples and the complete source code. To obtain the
full version, download the installer from http://simplemvvmtoolkit.codeplex.com/releases.

The CodePlex site includes documenation and a support forum, as well as video tutorials.
Refer to the Getting Started section of the online documentation:
http://simplemvvmtoolkit.codeplex.com/wikipage?title=Getting%20Started
